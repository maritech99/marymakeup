# Proyecto Web de Belleza

Este es un proyecto web desarrollado con HTML y publicado mediante GitHub Pages.

## Descripción

El sitio está orientado al ámbito del cuidado personal y la belleza, con el objetivo de organizar y presentar información relacionada con el cuidado de la piel y el bienestar general.

## Contenido del proyecto

El sitio incluye secciones informativas sobre:

- Rutinas de cuidado de la piel
- Tipos de piel y sus cuidados básicos
- Hábitos saludables relacionados con el bienestar
- Información general sobre ingredientes y productos de uso cosmético
- Ideas para futuras herramientas interactivas

## Objetivo del proyecto

El objetivo es desarrollar una base para un proyecto web que pueda evolucionar hacia una plataforma informativa más completa, con potencial de expansión a herramientas digitales y modelos de monetización.

## Tecnologías utilizadas

- HTML
- CSS básico
- GitHub Pages
